#### 选题八

  学生社团网站开发设计

   开发设计一个学生社团门户网站（针对一个具体的社团）。主要模块包括：社团介绍、加入社团、社团活动、公告、交流讨论等。主页提供新闻、通知公告等功能。前台可进行查询、浏览，后台提供各种信息的维护功能。
